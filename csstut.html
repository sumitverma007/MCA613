Selectors in CSS

In CSS we use selectors to target a particular element/s which we need to style.
Example of some selectors are:


Element Selector

Element selector are used to target a particular element of HTML .For example we're using Vietnam font throughout this page.This can be done by targeting the body element.

ID Selector

ID Selector Uses the id attribute of HTML element to select a specific element.This current para has some id and hence structured using this id.

Class Selector

Class selector select HTML element with specific class attribute.
