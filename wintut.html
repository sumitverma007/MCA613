INTRODUCTION

Windows Programming deals with the development of windows Applications.

C++ as a language from two perspective

  • Native ISO /ANSI C++ Windows application development.
  • Development of C++/CLI windows applications using windows Forms

C++ Applications

  • Native C++ Programs are those programs which runs natively on our computer.
  • We can also write C++ programs which runs under the control of CLR.These are called C++/CLI programs.

Windows Applications

  • Console Application-These are the command line character based programs.We can write this using either Win32 console application or through CLR console Application.
  • GUI BASED APPLICATION-This type of application can be written in following ways:
    • Using Windows Operating System API
    • Using Microsoft Foundation Class (MFC)
      •
    • Using Windwos Forms
